# 법률 도우미 ⚖️

> 일상 속 법률을 쉽게 검색하고, 매일 법률 상식을 받아보세요

## 기능

- **법률 검색**: 부동산, 교통, 세금, 파산, 특허 등 분야별 법률 검색
- **오늘의 법률 상식**: 매일 하나씩 알아두면 좋은 생활 법률 지식
- **메일링 서비스**: 매일 아침 8시 법률 상식 이메일 발송
- **Q&A 게시판**: AI가 법률 질문에 즉시 답변 (Gemini Flash)

## 기술 스택

- **Frontend**: Vanilla HTML/CSS/JS (GitHub Pages)
- **Backend**: Flask API (EC2)
- **AI**: Google Gemini 2.0 Flash
- **법률 DB**: 법제처 오픈 API (law.go.kr)

## 참고
